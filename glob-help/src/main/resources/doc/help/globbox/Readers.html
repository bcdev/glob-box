Supported Products in GlobToolbox 2.0

Version 2.0 of the BEAM GlobToolbox is capable of reading data products provided by multiple Glob* projects. The following table comprises the supported products and also the products which are not or not fully supported:

Project Product Type Support
GlobCover Bimonthly/Annual MERIS FR mosaics Supported
Global/Regional Land Cover Supported
GlobColour Global/DDS binned product Supported
Global/DDS mapped product Supported
MERIS L3 product Supported
GlobCarbon All types Will presumably be supported in GlobToolbox 2.1
ATSR World Fire ATSR2 and AATSR Supported
Medspiration L2 Pre-Processed (L2P), Analysed SST (L4) HR-DDS Data Supported
Global Land Cover 2000 GLC2000 Supported as GeoTIFF
GlobAerosol Individual-Sensor Orbit Aerosol All sensors supported except AATSR
Individual Global Aerosol Supported
Merged Global Aerosol Supported
Statistical Products Supported
IGBP Global LLC Supported

 

The products are opened by selecting the product type within File-> Import.
