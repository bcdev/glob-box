GlobToolbox

Introduction

About the BEAM GlobToolbox, Version 3.0

The BEAM GlobToolbox project has been brought into life in order to support users of data acquired by ESA's Glob* projects.

The GlobToolbox is developed by Brockmann Consult under contract to ESA.

Capabilities

The GlobToolbox 3.0 comprises the following features:
  • Read a large number of data products provided by multiple Glob* projects.
  • Create, edit, and view Time Series of these products (see VISAT -> Tools -> Time Series Tool -> Time Series Manager)
  • Analyse a Time Series using a dedicated graph tool (see VISAT -> Tools -> Time Series Tool -> Time Series Graph)
  • Playback of a Time Series using a dedicated Time Series player tool (see VISAT -> Tools -> Time Series Tool -> Time Series Player)
  • Export a Time Series as NetCDF/CF, as Google Earth KMZ-file (see VISAT -> Tools -> Time Series Tool -> Export Time Series as Google Earth KMZ file) or as animated gif
